Diagnosis Self 的一个例子,用nodejs readline mysql实现。



**Quick Start**  

数据库用的是MYSQL,先把.sql文件导入到你的数据库。

- 修改 config 的配置文件
- 打开你的cmd 或 其他命令行工具，cd 到这个目录
-  run  `npm install`
- run `node demo.js ` or `node demo1.js` , 他们只是提示稍微有些不同o(∩_∩)o 

----------

**怎么用呢**  
- 你可以直接输入数字开始自我诊断，但建议先输入 `list`  命令，先看看有哪些ID可以使用。
